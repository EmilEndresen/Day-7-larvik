Different tags

KBD

The kbd tag is used to define keyboard input, for example Ctrl + C to copy text
and is displayed in the browsers default monospace font

it is possible to achieve a richer effect by using css
Ctrl + V to paste

CODE

the code tag is used to define a piece of computer code
for example: The button tag defines a clickable button
the content is also displayed in the browsers default monospace font

We can also use css here to achieve a better effect
The button tag defines a clickable button

VAR

The var tag is used to define a variable in programming or a mathematical expression, and the content is usually displayed in italic, for example:

The area of a triangle is: 1/2 * b * h, where b is the base, and h is the vertical height

SAMP

The samp tag is used to define sample output from a computer program.
the content inside is displayed in the browsers default monospace font

For example:

File not found.
Press F1 to continue

DO NOT use these tags to display normal text!
